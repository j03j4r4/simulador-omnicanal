# simulador-omnicanal
Simulador web interactivo (HTML/JS) para modelar la rentabilidad omnicanal. Compara la rentabilidad de clientes omnicanal (Segmento C) vs. monocanal (A: Tienda, B: Online). Analiza métricas clave como CLV, Ratio CLV:CAC, costos de cumplimiento (BOPIS, Ship-from-Store) y Tasa de Retención. Herramienta educativa para retail y marketing.
